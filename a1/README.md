# Assignment 1 - HTML Text Formatting Tags

## Aim
To demonstrate various HTML text formatting tags (`<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<sub>`, `<sup>`, `<u>`, `<pre>`, `<q>`, `<code>`, `<abbr>`, `<kbd>`) inside a styled HTML table.

## Technologies Used
- HTML
- CSS (inline `<style>`)

## Steps Performed
1. Created `table.html` with a table listing common HTML text formatting tags.
2. Styled the table and page background using CSS gradients, hover effects, and alternating row colors.
3. Rendered each tag's live example in the third column for visual reference.

## Output
Open `table.html` in a browser to view the styled table of HTML text formatting tags with live examples.
